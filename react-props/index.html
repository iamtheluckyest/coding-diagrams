React - Understanding Props

Let's Start with JSX

JSX is a Javascript syntax developed by Facebook for use in React. Two important things about JSX for the purpose of this explanation.

  1. It looks like HTML, but it is Javascript. This means that while it looks like you are writing HTML tags with attributes, you are actually writing Javascipt functions that React turns into HTML.
  2. There is no JSX outside of React. Okay, actually there can be but JSX was specifically designed for React and remembering this will (hopefully) make understanding it a little easier.

What React does with JSX

When React sees Javascript written in JSX syntax, it breaks it apart and sticks the pieces into the React.createElement function. This function has takes three arguments

  1. Type. This is the type of element you are creating. It might be a regular HTML type (i.e. h1, a, p, div) or a component (i.e. Container, ListItem).
  2. Properties, or "props". Any "attributes" you give your "element" will be bundled together into a single object. If the type matches a regular HTML type, the props object will be interpreted as actual attributes for that element (src, href, name, alt, etc.). But if the type matches a component, the props object will be made available to that component via the props argument that the child component received as an argument.
  3. Children. These are the child "elements" of whatever element you are creating with React.createElement.

JSX

import React from "react";
import ColoredBox from "./components/ColoredBox";

const App = () => 
  <div>
    <img 
      src="http://logosrated.net/wp-content/uploads/parser/React-Logo-1.png" 
      alt="react logo"
    />
    <ColoredBox 
      boxColor="darkblue"
      textColor="white"
    />
  </div>
  
export default App;

Regular Javascript

(ignoring the import and export statements so we can focus on the JSX interpretation) 
var App = function App() {
  return React.createElement(
    "div",
    null,
    React.createElement("img", {
      src: "http://logosrated.net/wp-content/uploads/parser/React-Logo-1.png",
      alt: "react logo"
    }),
    React.createElement(ColoredBox, {
      boxColor: "darkblue",
      textColor: "white"
    })
  );
};
